# CycleLine web

Next.js 16 / React 19 / TypeScript frontend for the CycleLine Laravel API.

```bash
npm install
cp .env.example .env.local
npx auth secret
npm run dev
```

Validation:

```bash
npm run typecheck
npm run lint
npm run build
```

See the root [README](../README.md) for the complete setup and demo accounts.
